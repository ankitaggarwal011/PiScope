# ADS1015
*Interfacing the ADS1015 with Raspberry Pi*

---

## Instructions

1. Connect the **Adafruit ADS1015** to your **Raspberry Pi**

	**I2C Address :** `0x48`

	| Breakout Board | Raspberry Pi |
	|----------------|--------------|
	| `GND` | `GND` |
	| `5V` | `5V` |
	| `SDA` | `SDA` |
	| `SCL` | `SCL` |
	| `ADDR` | `GND` |

2. Check the devices connected to the I2C bus

 	```sh
		i2cdetect -y 1
	```
	
	*The ADS1015 should be listed.*
