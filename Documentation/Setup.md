
# Setup

<br>

## Hardware

1. Raspberry Pi B / B+ / 2

2. **[A Supported Breakout Board][Boards]**

<br>
<br>

## Software

1.  **[Python]**

    *Version `2.7`*
    
    <br>
    
2.  **Python Modules**
    
    - [`numpy`][Pip Numpy]
    
    - [`matplotlib`][Pip Matplot]
    
    - [`scipy`][Pip Scipy]
    
    - [`tkinter`][Tkinter]
    
    <br>
    
    These modules can be installed with:
    
    ```sh
    pip install     \
        matplotlib  \
        numpy       \
        scipy
    ```
    
<br>
